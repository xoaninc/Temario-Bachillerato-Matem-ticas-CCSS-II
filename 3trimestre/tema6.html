6.1 Concepto de funcion Selectividad

Funcion real de variable real

Una funcion es una correspondencia que a cada valor $x$ de un subconjunto de $\mathbb{R}$ (llamado dominio) le asigna un unico valor $y = f(x)$.

$$f: D \subseteq \mathbb{R} \to \mathbb{R}, \quad x \to y = f(x)$$

El valor $x$ se llama variable independiente y el valor $y = f(x)$ se llama variable dependiente o imagen de $x$.

Dominio y recorrido

El dominio ($\text{Dom}(f)$) es el conjunto de valores de $x$ para los que la funcion esta definida (tiene imagen).

El recorrido o imagen ($\text{Im}(f)$) es el conjunto de valores de $y$ que la funcion realmente toma.

Ejemplo: Para $f(x) = \sqrt{x}$, el dominio es $[0, +\infty)$ (no podemos hacer raiz de negativos) y el recorrido es $[0, +\infty)$ (la raiz siempre da valores positivos o cero).

Puntos de corte con los ejes Suele entrar

Son los puntos donde la grafica toca los ejes:

  • Corte con el eje X: hacemos $y = 0$ (es decir, $f(x) = 0$) y despejamos $x$.
  • Corte con el eje Y: hacemos $x = 0$ y calculamos $f(0)$.
Simetrias

Solo pueden existir si el dominio es simetrico respecto al origen (es decir, si $x$ esta en el dominio, $-x$ tambien).

  • Funcion par (simetrica respecto al eje Y): $f(x) = f(-x)$. Ejemplo: $f(x) = x^2$.
  • Funcion impar (simetrica respecto al origen): $f(x) = -f(-x)$. Ejemplo: $f(x) = x^3$.

6.2 Dominio de una funcion Selectividad

Clave para el examen

El dominio depende del tipo de funcion. Hay 3 restricciones basicas que debes buscar:

RestriccionQue mirasCondicion
No dividir por 0DenominadoresDenominador $\neq 0$
No raiz par de negativoRadicandos pares ($\sqrt{\ }$, $\sqrt[4]{\ }$...)Radicando $\geq 0$

Polinomica

Dominio de funciones polinomicas

$f(x) = a_0 + a_1 x + a_2 x^2 + \cdots + a_n x^n$

$$\text{Dom}(f) = \mathbb{R}$$

No hay restricciones: se puede sustituir cualquier valor de $x$.

Ejemplo: $f(x) = x^3 - 3x^2 + 6x + 2 \Rightarrow \text{Dom}(f) = \mathbb{R}$

Racional

Dominio de funciones racionales Suele entrar

$f(x) = \dfrac{P(x)}{Q(x)}$, con $P(x)$ y $Q(x)$ polinomios.

$$\text{Dom}(f) = \mathbb{R} \setminus \{x : Q(x) = 0\}$$

Quitamos los valores que anulan el denominador.

Ejemplo - Dominio racional

Halla el dominio de $f(x) = \dfrac{x^2 + 6x + 2}{x^2 - 5x + 6}$

Resolvemos $x^2 - 5x + 6 = 0$:

$$x = \frac{5 \pm \sqrt{25 - 24}}{2} = \frac{5 \pm 1}{2} \Rightarrow x = 3 \text{ o } x = 2$$

$\text{Dom}(f) = \mathbb{R} \setminus \{2, 3\}$

Dominio de funciones irracionales

Depende de si el indice de la raiz es par o impar:

Halla el dominio de $f(x) = \sqrt{x^2 - 5x + 6}$

Necesitamos $x^2 - 5x + 6 \geq 0$. Las raices son $x = 2$ y $x = 3$.

Exponencial

Dominio de funciones exponenciales

$f(x) = e^{P(x)}$ o $f(x) = a^{P(x)}$

$$\text{Dom}(f) = \text{Dom}(P)$$

La exponencial acepta cualquier exponente, asi que el dominio es el del exponente.

Ejemplo: $f(x) = 5^{3x+1} \Rightarrow \text{Dom}(f) = \mathbb{R}$

Cuidado: $f(x) = e^{2/(x^2-5x+6)}$. Hay una division en el exponente, asi que $x \neq 2, 3$. Dominio $= \mathbb{R} \setminus \{2, 3\}$.

Logaritmica

Dominio de funciones logaritmicas Suele entrar

$f(x) = \log_a(P(x))$

$$\text{Dom}(f) = \{x \in \mathbb{R} : P(x) > 0\}$$

El argumento del logaritmo debe ser estrictamente positivo (mayor que cero, no vale cero).

Ejemplo - Dominio logaritmico

Halla el dominio de $f(x) = \ln\!\left(\dfrac{x+5}{x^2-4x+3}\right)$

Necesitamos $\dfrac{x+5}{x^2-4x+3} > 0$.

Raices: $x + 5 = 0 \Rightarrow x = -5$; $\quad x^2 - 4x + 3 = 0 \Rightarrow x = 1, x = 3$.

6.3 Limites de funciones Selectividad

Idea intuitiva de limite

El limite de $f(x)$ cuando $x$ tiende a $x_0$ es el valor al que se acercan las imagenes ($y$) cuando las $x$ se acercan a $x_0$.

No importa lo que valga $f(x_0)$ ni si existe: solo importa a que valor se aproxima la funcion alrededor de $x_0$.

Ejemplo: $f(x) = x^2$, $\lim_{x \to 2} x^2 = 4$. Si $x$ se acerca a 2 (por ejemplo: $1{,}9 \to 3{,}61$; $1{,}99 \to 3{,}96$; $2{,}01 \to 4{,}04$...), $f(x)$ se acerca a 4.

Limites laterales

Limites laterales Suele entrar

Podemos acercarnos a $x_0$ por la izquierda (valores menores) o por la derecha (valores mayores):

  • Limite por la izquierda: $\displaystyle\lim_{x \to x_0^-} f(x)$
  • Limite por la derecha: $\displaystyle\lim_{x \to x_0^+} f(x)$

El limite existe si y solo si ambos limites laterales existen y son iguales:

$$\exists \lim_{x \to x_0} f(x) \iff \lim_{x \to x_0^-} f(x) = \lim_{x \to x_0^+} f(x)$$
Ejemplo - Limites laterales en funcion a trozos

$f(x) = \begin{cases} x^2 & \text{si } x > 2 \\ 4 & \text{si } x < 2 \end{cases}$. Calcula $\lim_{x \to 2} f(x)$.

Por la derecha ($x > 2$): $\lim_{x \to 2^+} x^2 = 4$

Por la izquierda ($x < 2$): $\lim_{x \to 2^-} 4 = 4$

Como coinciden, $\lim_{x \to 2} f(x) = 4$.

El limite vale 4 aunque la funcion no tenga imagen en $x = 2$ (no esta definida ahi).

Calculo directo de limites

Si $f$ es una funcion elemental (polinomica, racional, exponencial, logaritmica...) y esta definida en $x = a$, entonces:

$$\lim_{x \to a} f(x) = f(a)$$

Es decir, simplemente sustituimos $x$ por $a$.

Ejemplo: $\lim_{x \to 2} (x^2 + x - 1) = 4 + 2 - 1 = 5$

Indeterminaciones

Las 4 indeterminaciones de la asignatura Siempre entra

Cuando al sustituir obtenemos una de estas formas, hay que operar para resolver:

$$\frac{0}{0}, \quad \frac{\infty}{\infty}, \quad \infty - \infty, \quad 0 \cdot \infty$$
Indeterminacion $\frac{0}{0}$ — Cociente de polinomios Muy frecuente

$\displaystyle\lim_{x \to a} \frac{P(x)}{Q(x)} = \frac{0}{0}$

Metodo: Factorizamos ambos polinomios por Ruffini (dividiendo por $(x - a)$) y simplificamos.

Ejemplo - Indeterminacion 0/0

$\displaystyle\lim_{x \to 1} \frac{x^2 - 2x + 1}{x^2 - 1}$

Sustituyendo: $\frac{1 - 2 + 1}{1 - 1} = \frac{0}{0}$ indeterminado.

Factorizamos:

$$\frac{(x-1)^2}{(x-1)(x+1)} = \frac{x-1}{x+1}$$

Ahora sustituimos: $\dfrac{1-1}{1+1} = \dfrac{0}{2} = 0$

Ejemplo - Indeterminacion 0/0 con radicales

$\displaystyle\lim_{x \to 0} \frac{x}{1 - \sqrt{1-x}}$

Sustituyendo: $\frac{0}{1-1} = \frac{0}{0}$. Multiplicamos por el conjugado:

$$\frac{x}{1 - \sqrt{1-x}} \cdot \frac{1 + \sqrt{1-x}}{1 + \sqrt{1-x}} = \frac{x(1 + \sqrt{1-x})}{1 - (1-x)} = \frac{x(1 + \sqrt{1-x})}{x}$$

Simplificamos la $x$:

$$\lim_{x \to 0} (1 + \sqrt{1-x}) = 1 + \sqrt{1} = 2$$
Indeterminacion $\frac{\infty}{\infty}$ — Cociente de polinomios Muy frecuente

$\displaystyle\lim_{x \to \infty} \frac{P(x)}{Q(x)} = \frac{\infty}{\infty}$

Metodo: Nos quedamos SOLO con el monomio de mayor grado del numerador y del denominador.

Si...Resultado
grado numerador $<$ grado denominador$0$
grado numerador $>$ grado denominador$\pm\infty$
mismo gradocociente de los coeficientes principales
Ejemplo - Tres casos de $\infty/\infty$

a) $\displaystyle\lim_{x \to \infty} \frac{2x+1}{x^2+3x+9}$   b) $\displaystyle\lim_{x \to \infty} \frac{3x^2-4x+1}{6x+9}$   c) $\displaystyle\lim_{x \to \infty} \frac{4x^2-4x+1}{6x^2+x+8}$

a) Grado 1 vs grado 2 $\Rightarrow$ grado denominador mayor $\Rightarrow \displaystyle\lim = \frac{2x}{x^2} = \frac{2}{x} = 0$

b) Grado 2 vs grado 1 $\Rightarrow$ grado numerador mayor $\Rightarrow \displaystyle\lim = \frac{3x^2}{6x} = \frac{x}{2} = \infty$

c) Mismo grado (2) $\Rightarrow \displaystyle\lim = \frac{4x^2}{6x^2} = \frac{4}{6} = \frac{2}{3}$

Indeterminacion $\infty - \infty$

Dos casos principales:

  • Con funciones racionales: ponemos a comun denominador y obtenemos $\frac{0}{0}$ que resolvemos por Ruffini.
  • Con radicales: multiplicamos y dividimos por el conjugado y obtenemos $\frac{\infty}{\infty}$.
Ejemplo - $\infty - \infty$ con racionales

$\displaystyle\lim_{x \to 2}\left(\frac{1}{x-2} - \frac{2x}{x^2-4}\right)$

Comun denominador ($x^2 - 4 = (x-2)(x+2)$):

$$\frac{(x+2) - 2x}{x^2-4} = \frac{2-x}{x^2-4} = \frac{-(x-2)}{(x-2)(x+2)} = \frac{-1}{x+2}$$

Sustituimos: $\dfrac{-1}{2+2} = \dfrac{-1}{4}$

Ejemplo - $\infty - \infty$ con radicales

$\displaystyle\lim_{x \to \infty}\left(\sqrt{3x+2} - \sqrt{3x-1}\right)$

Multiplicamos por el conjugado:

$$\frac{(\sqrt{3x+2} - \sqrt{3x-1})(\sqrt{3x+2} + \sqrt{3x-1})}{\sqrt{3x+2} + \sqrt{3x-1}} = \frac{(3x+2) - (3x-1)}{\sqrt{3x+2} + \sqrt{3x-1}} = \frac{3}{\sqrt{3x+2} + \sqrt{3x-1}}$$

Cuando $x \to \infty$, el denominador $\to \infty$, asi que el limite es $\dfrac{3}{\infty} = 0$.

Indeterminacion $0 \cdot \infty$

Metodo: Transformamos en $\frac{0}{0}$ o $\frac{\infty}{\infty}$ reescribiendo el producto como cociente.

Ejemplo - $0 \cdot \infty$

$\displaystyle\lim_{x \to \infty} \frac{1}{x} \cdot \sqrt{x^2 + 3x}$

Reescribimos: $\dfrac{\sqrt{x^2+3x}}{x} = \dfrac{\infty}{\infty}$

Nos quedamos con el monomio de mayor grado bajo la raiz: $\dfrac{\sqrt{x^2}}{x} = \dfrac{x}{x} = 1$

Limites tipo $k/0$ Suele entrar

Cuando al sustituir obtenemos $\frac{k}{0}$ con $k \neq 0$, el limite es $\pm\infty$.

Para saber el signo, hay que estudiar los limites laterales: ver si el denominador tiende a $0^+$ o $0^-$.

  • Si los laterales coinciden $\Rightarrow$ el limite es $+\infty$ o $-\infty$.
  • Si los laterales NO coinciden $\Rightarrow$ no existe el limite (hay asintota vertical).
Ejemplo - Limite tipo $k/0$ con laterales

$\displaystyle\lim_{x \to 1} \frac{x+1}{x-1}$

Sustituyendo: $\frac{2}{0}$. Hay que estudiar laterales.

Por la derecha ($x \to 1^+$): $x - 1 \to 0^+$ (positivo), asi que $\frac{2}{0^+} = +\infty$

Por la izquierda ($x \to 1^-$): $x - 1 \to 0^-$ (negativo), asi que $\frac{2}{0^-} = -\infty$

Los laterales no coinciden $\Rightarrow$ $\nexists\lim_{x \to 1} \frac{x+1}{x-1}$.

Limite de funciones a trozos Suele entrar

En los puntos de union entre los trozos, hay que calcular los limites laterales usando la formula de cada trozo:

  • Limite por la izquierda: usamos la formula del trozo de la izquierda.
  • Limite por la derecha: usamos la formula del trozo de la derecha.
  • Si coinciden, el limite existe. Si no, no existe.
Ejemplo - Limite en funcion a trozos

$f(x) = \begin{cases} 1 & \text{si } x < -1 \\ x^2 & \text{si } -1 \leq x < 1 \\ 2 & \text{si } x \geq 1 \end{cases}$. Estudia los limites en $x = -1$ y $x = 1$.

En $x = -1$:

$\lim_{x \to -1^-} 1 = 1$   $\lim_{x \to -1^+} x^2 = 1$

Coinciden $\Rightarrow \lim_{x \to -1} f(x) = 1$ ✓

En $x = 1$:

$\lim_{x \to 1^-} x^2 = 1$   $\lim_{x \to 1^+} 2 = 2$

No coinciden $\Rightarrow \nexists\lim_{x \to 1} f(x)$

Funcion a trozos: ● incluido, ○ no incluido

En $x=-1$ el limite existe (coinciden). En $x=1$ no existe (salto de 1 a 2).

6.4 Asintotas Selectividad

Que son las asintotas

Las asintotas son rectas a las que la funcion se va acercando indefinidamente. Hay tres tipos:

Ejemplo visual: los 3 tipos de asintotas
Asintota vertical (A.V.) Suele entrar

Recta de ecuacion $x = a$. Se busca donde el denominador se anula (o en los bordes del dominio):

$$\lim_{x \to a} f(x) = \pm\infty \Rightarrow x = a \text{ es A.V.}$$

Para saber la posicion de la curva respecto a la asintota, calculamos los limites laterales.

Asintota horizontal (A.H.) Suele entrar

Recta de ecuacion $y = b$. Se busca haciendo:

$$\lim_{x \to +\infty} f(x) = b \quad \text{y/o} \quad \lim_{x \to -\infty} f(x) = b$$

Si alguno de esos limites da un numero finito $b$, entonces $y = b$ es A.H.

Asintota oblicua (A.O.) No selectividad

Recta de ecuacion $y = mx + n$. Solo existe si no hay A.H. en esa direccion.

$$m = \lim_{x \to \infty} \frac{f(x)}{x}, \qquad n = \lim_{x \to \infty} \big(f(x) - mx\big)$$

Si $m$ existe y es finito (y $\neq 0$), y $n$ existe, entonces hay A.O.

Ejemplo - Asintotas de una funcion racional

Halla las asintotas de $f(x) = \dfrac{x^2 + 2x}{x + 1}$

A.V.: $x + 1 = 0 \Rightarrow x = -1$. Comprobamos: $\lim_{x \to -1} f(x) = \frac{-1}{0} = \pm\infty$ ✓

A.H.: $\lim_{x \to \infty} \frac{x^2+2x}{x+1} = \frac{x^2}{x} = x = \infty$. No hay A.H.

A.O.: Dividimos: $\frac{x^2+2x}{x+1} = x + 1 + \frac{-1}{x+1}$

La asintota oblicua es $y = x + 1$.

Cuando $x \to +\infty$: $\frac{-1}{x+1} < 0 \Rightarrow$ la curva esta por debajo de la asintota.

Cuando $x \to -\infty$: $\frac{-1}{x+1} > 0 \Rightarrow$ la curva esta por encima de la asintota.

6.5 Continuidad Selectividad

Continuidad en un punto Siempre entra

Una funcion $f(x)$ es continua en $x = a$ si se cumplen las 3 condiciones:

#CondicionSignificado
1$\exists\, f(a)$La funcion tiene imagen en $a$ ($a$ esta en el dominio)
2$\exists\, \lim_{x \to a} f(x)$El limite existe (los laterales coinciden)
3$\lim_{x \to a} f(x) = f(a)$El limite coincide con el valor de la funcion

Importante: En el examen hay que escribir las 3 condiciones y comprobar cada una. Es la definicion de continuidad en un punto.

Funciones continuas en su dominio

Las funciones elementales son continuas en todos los puntos de su dominio:

  • Polinomicas: continuas en todo $\mathbb{R}$.
  • Racionales: continuas en $\mathbb{R}$ excepto donde se anula el denominador.
  • Irracionales, exponenciales, logaritmicas: continuas en su dominio.

Solo hay que estudiar la continuidad manualmente en las funciones a trozos, en los puntos donde cambia la definicion.

Ejemplo - Continuidad de funcion a trozos

Estudia la continuidad de $f(x) = \begin{cases} x^2 & \text{si } x < 2 \\ 4 & \text{si } x \geq 2 \end{cases}$ en $x = 2$.

1. $f(2) = 4$ ✓ (existe, porque $x = 2$ esta en el trozo $x \geq 2$)

2. $\lim_{x \to 2^+} f(x) = \lim_{x \to 2^+} 4 = 4$

$\quad\;\; \lim_{x \to 2^-} f(x) = \lim_{x \to 2^-} x^2 = 4$

$\quad\;\;$ Coinciden $\Rightarrow \lim_{x \to 2} f(x) = 4$ ✓

3. $\lim_{x \to 2} f(x) = 4 = f(2)$ ✓

Se cumplen las 3 condiciones $\Rightarrow$ $f$ es continua en $x = 2$.

Continua vs discontinua — ● punto incluido, ○ punto NO incluido

Discontinuidad

Tipos de discontinuidad

Si alguna de las 3 condiciones falla, la funcion es discontinua en ese punto:

Condicion que fallaTipoEjemplo visual
No existe $f(a)$ pero si el limiteEvitable (un "agujero")Se puede "rellenar" definiendo $f(a) = L$
Los limites laterales existen pero no coincidenDe saltoLa grafica "salta" de un valor a otro
Algun limite lateral es $\pm\infty$Esencial (asintota vertical)La grafica se va al infinito
Ejemplo - Discontinuidad evitable

$f(x) = \begin{cases} x^2 & \text{si } x < 2 \\ 4 & \text{si } x > 2 \end{cases}$ (no definida en $x = 2$).

1. $f(2)$ no existe ✗ $\Rightarrow$ ya es discontinua.

Pero $\lim_{x \to 2^-} x^2 = 4 = \lim_{x \to 2^+} 4$, asi que el limite si existe y vale 4.

Es una discontinuidad evitable: si definieramos $f(2) = 4$, seria continua.

Ejemplo - Hallar parametros para continuidad

Halla $a$ y $b$ para que $f(x) = \begin{cases} x+3 & \text{si } x < 0 \\ ax+b & \text{si } 0 \leq x \leq 2 \\ 0{,}5x+1 & \text{si } x > 2 \end{cases}$ sea continua en $\mathbb{R}$.

Cada trozo es polinomico, asi que solo estudiamos los puntos de union: $x = 0$ y $x = 2$.

En $x = 0$: Continuidad $\Rightarrow$ limites laterales iguales y $= f(0)$.

$\lim_{x \to 0^-}(x+3) = 3$   $f(0) = a \cdot 0 + b = b$

$\Rightarrow b = 3$

En $x = 2$:

$f(2) = 2a + b = 2a + 3$   $\lim_{x \to 2^+}(0{,}5x+1) = 2$

$\Rightarrow 2a + 3 = 2 \Rightarrow a = -\frac{1}{2}$

Solucion: $a = -1/2$, $b = 3$.
